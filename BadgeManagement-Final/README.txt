Badge Management Application
==========================

How to Run:
1. Ensure Node.js is installed (download from https://nodejs.org/)
2. Double-click 'Simple-Start.bat'
3. The application will open in your default browser

First Time Setup:
- The first time you run it, it will install required modules (internet connection required)
- This might take a minute

Troubleshooting:
- If you see errors, make sure Node.js is installed correctly
- Ensure port 3000 is available
- Run 'Simple-Start.bat' as administrator if you encounter permission issues

To distribute:
1. Zip this entire folder
2. Send the ZIP file to users
3. They just need to extract and run 'Simple-Start.bat'
